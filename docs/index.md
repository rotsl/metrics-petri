# Metrics Petri

Metrics Petri segments fungal colonies in Petri-dish photographs and converts the masks
into physical growth and morphology measurements.

## Start here

- Read the [user guide](user-guide.md) for installation and command-line usage.
- See the [batch pipeline guide](batch-pipeline.md) for metadata and outputs.
- Review the [model card](model-card.md) before interpreting segmentation results.
- Browse the [API reference](api/index.md), generated from Python docstrings and
  signatures with mkdocstrings.
